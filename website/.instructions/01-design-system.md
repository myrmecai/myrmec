# Myrmec Design System Spec
**Role:** Foundation & Layout

## 1. Visual Identity
- **Aesthetic:** Enterprise SaaS / "Control Plane" aesthetic. Clean, high-density, professional.
- **Theme:** Strict Dark Mode.
- **Palette:**
  - Background: Slate-950 (Deep Obsidian)
  - Surface: Slate-900 (Card backgrounds)
  - Primary: Indigo-500 to Blue-600 (Action buttons)
  - Success: Emerald-500 (Active status indicators)
  - Border: Slate-800 (Subtle dividers)

## 2. Typography
- **Headings:** Inter or Geist Sans (Bold, tight tracking)
- **Body:** Inter (Regular, legible)
- **Data/Code:** JetBrains Mono (For model IDs, logs, and CLI commands)

## 3. Layout Structure
- **Navigation:** Top-bar with "Myrmec" logo, and Tagline".
- **Sidebar (Admin View):**
  - Dashboard
  - Model Registry
  - Agent Profiles
  - Workflow Sutras
  - Audit Logs
- **Responsiveness:** Mobile-first, but optimized for desktop "Management" views.
