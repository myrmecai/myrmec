## Website deployment

This website is statically generated by Astro.

```bash
npm install
npm run build
```

Serve the dist directory using nginx or any static host.

## Platform deployment notes

- Run Control Plane Engine behind a load balancer
- Run agents as independent containers with registration keys
- Use managed PostgreSQL for production durability
- Apply observability for logs, traces, and metrics

## Scaling strategy

- Horizontally scale stateless engine instances
- Scale agent pools by profile and workload type
- Use separate model routes per project requirements
